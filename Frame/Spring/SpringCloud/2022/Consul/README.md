# Consul

Consul 是一套开源的分布式服务发现和配置管理系统，由 HashiCorp 公司用 Go 语言开发。部署起来非常容易，只需要极少的可执行程序和配置文件，具有绿色、轻量级的特点。

**官方网站**：http://www.consul.io
